`Input` — text field on an inset well. `mono` sets the value in the values typeface (ids, ports, paths, search); `accent` tints the focus ring relay (default) / provider / tool. Supports `iconLeft` and a trailing `affix`.

```jsx
<Input iconLeft={searchIcon} placeholder="Search 312 models…" />
<Input mono placeholder="claude-sonnet-4-6" affix="⏎" />
<Input accent="provider" placeholder="Provider name" />
```
